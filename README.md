# MinecraftTunnel
Minecraft 转发程序
高效率的 Minecraft 协议转发\n
可支持自定义 Motd , 自定义 转发目的

.net runtime : 
Centos   : https://docs.microsoft.com/en-us/dotnet/core/install/linux-centos\n
Windows  : 自行观看 

示例 ：\n
Centos 7 :\n
rpm -Uvh https://packages.microsoft.com/config/centos/7/packages-microsoft-prod.rpm \n
yum install dotnet-runtime-5.0 -y \n
yum install unzip -y \n
yum install screen -y \n

Windows 7 ： \n
先安装 VC++2015 - 2019 \n
然后安装  KB2533623 补丁即可 \n
