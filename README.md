# MinecraftTunnel
Minecraft 转发程序

高效率的 Minecraft 协议转发

可支持自定义 Motd , 自定义 转发目的

.net runtime : 

Centos   : https://docs.microsoft.com/en-us/dotnet/core/install/linux-centos

Windows  : 自行观看 

示例 ：
Centos 7 :

rpm -Uvh https://packages.microsoft.com/config/centos/7/packages-microsoft-prod.rpm

yum install dotnet-runtime-5.0 -y

yum install unzip -y

yum install screen -y

Windows 7 ：
先安装 VC++2015-2019 
然后安装  KB2533623 补丁即可
